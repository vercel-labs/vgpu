# `orbit`

`orbit(time, options?)` returns a column-major model matrix that rotates around the Y axis. Use it when an example needs deterministic motion without a scene graph.

```ts
import { orbit } from "vgpu/scene";

const model = orbit(gpu.time, { radius: 2, height: 0.5, speed: 0.4 });
draw.set({ model });
```

Options:

- `radius`: orbit radius. Defaults to `1`.
- `height`: Y translation. Defaults to `0`.
- `speed`: angular speed multiplier. Defaults to `1`.
