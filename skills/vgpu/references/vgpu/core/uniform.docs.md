# `Uniform`

Ring-0 escape hatch for a user-owned uniform buffer. Prefer `set({ value })` for ordinary values; use `Uniform` when one write must feed many draws or when you need byte-level control.

```ts
import { Uniform } from "vgpu/core";
const camera = new Uniform(gpu.device, { size: 64, label: "camera" });
const cube = gpu.draw({ shader: LIT_WGSL });
cube.set({ camera });
camera.write(viewProjection);
```

Because `camera` is a resource, ownership is user-owned from the first `set()` (R1). vgpu only binds it; you own writes and lifetime.
